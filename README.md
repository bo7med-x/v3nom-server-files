## 🎮 What's Inside v4.0

* **The New Custom UI:** The fully redesigned modern layout with vibrant, high-contrast menus.
* **Audio Replacement Pack:** Replaces default game sounds with clean, customized audio for better feedback.
* **Dynamic Head Icons:** Custom tags that automatically display player ranks (Admin, Guest, or specific Member Ranks) directly above the player's head upon login.
* **Custom Map Pack:** Includes a selection of new maps ready to play on the V3NOM server.
* **Automated Installer:** Wrapped into a clean `.exe` setup that automatically detects your game directory and places files in the `main` folder.
